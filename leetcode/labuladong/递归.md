### 遍历～回溯
全排列等(1,2,3): [1,2,3][1,3,2][2,3,1][2,1,3][3,1,2][3,2,1]
```python
result = []
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return
    
    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选
```
